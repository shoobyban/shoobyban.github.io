Toddlers’ Blocks for iPad and iPhone

 23/01/2013 -  Sam Ban

Finally I’ve released my block-building game for the kids! They’ve been testing it on the iPad for ages, asking for more faces, telling me that they like this, don’t like that etc.

Of course for App Store screen-shots I had to modify the UI, for loading and saving block layouts, it’s hard to create the same screen-shots for all the resolutions… and left a broken callback in the code even after testing, so 1.0 has a bug that you click on the menu and click again, boom, it crashes. I’m not really happy about it, but nobody is prefect…

Fix is already waiting for review, I hope I will be lucky to have it reviewed very soon.
